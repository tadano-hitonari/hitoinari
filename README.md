## 最大スタック


# 回避された時

キラーバイティング自動反撃。

処理位置

```erb
固有行動直後処理_キャラ_2845
```

入口

```erb
SIF 回避反応者:0 < 0
    RETURN 0
```

---

# キラーバイティング

RETURNは必ず

```erb
RETURN 消費ＭＰ
```

とすること。

単なる

```erb
RETURN
```

では再使用可能になる不具合が発生する。

# 開発ルール

## 実装方針

- 存在しない識別子を勝手に作らない。
- 実際のテンプレート・既存ERB・有識者情報を最優先とする。
- 推測で `FLAG:` や `CFLAG:` を新設しない。

## 刻印テンプレート

```erb

```

で行う。
